Enjoy the forest!

Animated forest scene

One of my latest project is about the forest scene that can be used as a game background. The leaves are generated by the user's interaction, there is also a wind function and a rain effect. The code was written in processing software using Java language

Instructions:

Press 'g' key to generate the leaves

Press 'l' key to generate the wind to the left

Press 'r' key to generate the wind to the right
